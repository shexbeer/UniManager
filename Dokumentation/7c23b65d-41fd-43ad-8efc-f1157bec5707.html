  Anmeldung zum Leistungsnachweis


User Stereotype 
IntentDer Student möchte die Zulassung zum Leistungsnachweis erwirken.
DescriptionDas System zeigt dem Benutzer eine Liste aller LN, die zu den Modulen seines SG gehören, die er dann einzeln anmelden kann. Bei unzulässigen Anmeldungsversuchen weißt das System den Benutzer durch Fehlermeldungen auf das Problem hin. Unzulässig sind bei schriftlichen und mündlichen Prüfungen Anmeldungen außerhalb der Fristen für Sommer- oder Wintersemester , ebenso Anmeldungen zu LN die bereits erfolgreich absolviert wurden. Das Anmeldedatum wird vom System gespeichert. Benötigte Informationen: Matrikelnummer, PrüfungsID, ModulID
Pre ConditionsEin Leistungsnachweis muss angeboten werden.
Post ConditionsAnmeldung erfolgreich.
Alternate CoursesDie Anmeldung erfolgt nicht rechtzeitig ( könnte vlt nach Absprache noch vom Lehrenden nachgetragen werden )
